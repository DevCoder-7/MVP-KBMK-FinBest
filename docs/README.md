# Dokumentasi FinBest MVP Web

Folder ini menyimpan dokumentasi teknis yang dibutuhkan untuk development, QA, dan deployment.

## Isi

- `DEPLOYMENT.md` - panduan lokal, Vercel, dan opsi database produksi.
- `MARKET_DATA.md` - penjelasan provider market data, cache, fallback, dan env.
- `REPO_STRUCTURE.md` - peta folder repo dan aturan mana yang boleh/tidak boleh di-commit.
- `QA_CHECKLIST.md` - checklist regresi sebelum demo, push, atau deploy.

## Catatan Archive

`docs/archive/` dipakai untuk artifact lokal seperti hasil ekstraksi prototype, worklog, screenshot QA, atau material sementara. Folder ini di-ignore dari Git supaya repo tetap ringan.
